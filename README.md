# formalidade-informalidade-brasil-pnadc-2024

Este repositório contém os códigos utilizados na extração, tratamento e análise
estatística dos microdados da Pesquisa Nacional por Amostra de Domicílios Contínua (PNADC),
com foco na comparação entre trabalhadores formais e informais no Brasil.

## Ferramentas
- Google BigQuery
- Python (Google Colab)

## Base de dados
- PNADC (IBGE), com acesso via Base dos Dados

## Observação
Os códigos têm finalidade acadêmica e visam permitir a reprodutibilidade
dos resultados apresentados no trabalho.
